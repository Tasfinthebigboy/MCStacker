MCStacker for Minecraft Java Edition 1.20.5/6

Welcome to MCStacker for Minecraft 1.20.5/6. This website will help you generate many commands for Minecraft Java Edition. The syntax for commands containing items changed significantly in this version. Fortuanately, MCStacker can convert most of your 1.20.4 style commands and output them in the new format. Most of the specific options available for entities, items and blocks can be generated with this tool. MCStacker began in 2014 and many new features have been added regularly ever since. I continue to implement new functionality and keep it up-to-date to the latest version of Minecraft.
